## 简介
此插件是为sublime快速生成thinkphp的代码块而开发的；其实甚至称不上是插件；只是就是2个文件；如果你用sublime做thinkphp开发；那么看到此插件；你绝对有种相见恨晚的感觉；更甚至；只要你是sublime的用户；都会感激流涕的；
